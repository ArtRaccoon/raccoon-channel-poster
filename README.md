# raccoon-channel-poster

Telegram posting bot (aiogram 3.x, SQLite, APScheduler).

## Доступно сейчас в UI
- Добавление и управление своими каналами.
- Создание черновиков постов (текст/фото).
- Публикация постов в выбранный канал.
- Ручное планирование постов по дате/времени.
- Просмотр черновиков и последних постов.
- Админ-статистика.
- Админ-кнопка `💾 Бэкап базы` (SQLite backup API).
- Админ-кнопка `🚨 Ошибки` (последние ошибки публикации из `publish_logs`).

## Надёжность и миграции
- Безопасные миграции SQLite через `PRAGMA table_info` + `ALTER TABLE ADD COLUMN`.
- Существующие данные не удаляются.
- `.env` не коммитится.
- Игнорируются артефакты БД и бэкапов: `*.db`, `backups/`, `backups/*.db`.

## Публикация
- Единый helper публикации: `bot/services/publishing.py`.
- Ручная публикация и scheduler используют общий `publish_post`.
- Есть валидация перед публикацией, включая запрет пустого text-поста.

## Подготовлено в базе для следующих PR
- URL-кнопки.
- Автоподписи каналов.
- Альбомы.
- Шаблоны подписей.

## systemd update flow
1. Pull latest branch.
2. Install/update dependencies.
3. Restart bot service.
4. Check `logs/bot.log`.

## Proxy
- Общий `PROXY_URL=socks5://127.0.0.1:1080` через `AiohttpSession` (логика не менялась).
