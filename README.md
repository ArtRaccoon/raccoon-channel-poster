# raccoon-channel-poster

Telegram-бот для публикации текстовых постов в каналы пользователей.

## Что делает бот

- Пользователь добавляет канал, где он администратор.
- Создаёт текстовый пост.
- Публикует пост в один из своих каналов.
- Владелец из `OWNER_IDS` видит админ-панель со статистикой.

## Настройка `.env`

Скопируйте шаблон:

```bash
cp .env.example .env
```

Заполните:

- `BOT_TOKEN` — токен бота.
- `OWNER_IDS` — список id владельцев через запятую.
- `PROXY_URL` — например `socks5://127.0.0.1:1080`.
- `TIMEZONE`, `DATABASE_PATH`, `LOG_LEVEL`.

`.env` не должен попадать в git.

## Запуск локально

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
./run.sh
```

## Использование общего sing-box прокси

Бот поддерживает общий SOCKS5 прокси через `AiohttpSession` (`aiogram 3.x`).

Пример:

```env
PROXY_URL=socks5://127.0.0.1:1080
```

Если `PROXY_URL` пустой — бот работает в direct-режиме.

Важно: второй sing-box поднимать не нужно, используется уже поднятый общий прокси.

## Как добавить бота в канал

1. Добавьте бота в канал.
2. Выдайте боту права администратора.
3. Разрешите публикацию сообщений.
4. В боте нажмите `➕ Добавить канал` и отправьте `@username` канала.

Необходимые права бота: администратор с возможностью публиковать сообщения.
