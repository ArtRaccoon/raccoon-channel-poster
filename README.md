# raccoon-channel-poster

Telegram posting bot (aiogram 3.x, SQLite, APScheduler).

## Главное меню
- ✍️ Создать пост
- ⚙️ Настройки
- 🛠 Админ-панель (только для OWNER_IDS)

## Создать пост
- 🖼 Один пост
- 🗓 Расписание
- ⬅️ Назад

### Один пост
Полностью работает:
- выбор активного канала;
- загрузка текста / одиночного фото / grouped media (альбом);
- создаётся один draft-пост;
- компактная панель действий;
- предпросмотр только по кнопке 👁.

### Расписание
Полностью работает:
- выбор канала;
- показ текущего расписания (или дефолтного шаблона);
- редактирование расписания в формате `Пн: HH:MM, ...`;
- создание batch-сессии;
- сбор контента (текст, одиночные фото, альбомы) без спама preview;
- завершение словом `готово`;
- распределение постов по ближайшим слотам и установка `scheduled_at`.

## Настройки
Работают разделы:
- 📣 Каналы: список, отключение, проверка доступа (добавление — через существующий flow `➕ Добавить канал`);
- 🗓 Расписание каналов: выбрать канал, посмотреть/изменить/очистить расписание;
- ✍️ Автоподпись: выбрать канал, посмотреть, изменить, очистить через `-` или `off`.

## Миграции
- Безопасные миграции SQLite через `PRAGMA table_info` + `ALTER TABLE ADD COLUMN`.
- `post_batches` создаётся через `CREATE TABLE IF NOT EXISTS`.
- В `posts` поддерживается `batch_id`.

## Proxy
- `PROXY_URL` и `AiohttpSession` не менялись.
