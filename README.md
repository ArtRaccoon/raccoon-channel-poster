# raccoon-channel-poster

Telegram posting bot (aiogram 3.x, SQLite, APScheduler).

## Новый UX
- Главное меню: добавление канала, мои каналы, создание поста, пакетное планирование, черновики, настройки, помощь.
- Раздел **«Мои каналы»** показывает только список каналов без технических полей.
- Раздел **«Настройки»** вынесен отдельно: автоподпись, расписание, кнопки канала.

## Создание поста
1. Пользователь выбирает канал.
2. Отправляет текст, одиночное фото или grouped media.
3. Бот показывает preview и меню действий (редактирование, замена медиа, автоподпись, публикация сейчас/позже, очередь, сохранение в черновик).

### Автоподпись
- На уровне поста есть флаг `use_signature`.
- По умолчанию включён.
- Если выключить, подпись канала не добавляется.

## Медиа
- text -> `send_message`
- photo -> `send_photo`
- album -> `send_media_group`
- Лимиты Telegram:
  - caption: 1024
  - text: 4096
- Если подпись + текст превышают лимит, публикация не выполняется.

## Планирование контента
- Кнопка **🗓 Запланировать контент** — отдельный batch-режим (MVP).
- Обычный режим **✍️ Создать пост** остаётся для одного поста.
- Grouped media обрабатывается как один album-пост.

## Надёжность и миграции
- Безопасные миграции SQLite через `PRAGMA table_info` + `ALTER TABLE ADD COLUMN`.
- Добавлены поля posts: `use_signature`, `repeat_enabled`, `repeat_interval_minutes`, `repeat_until`.
- Поддерживается fresh DB и существующие БД.

## systemd update flow
1. `git pull`
2. Обновить зависимости
3. Перезапустить systemd unit
4. Проверить `logs/bot.log`
