# raccoon-channel-poster

Telegram posting bot (aiogram 3.x, SQLite, APScheduler).

## Главное меню
- ✍️ Создать пост
- ⚙️ Настройки
- 🛠 Админ-панель (только для `OWNER_IDS`)

## Создать пост
- 🖼 Один пост
- 🗓 Расписание

### Один пост
1. Выбор канала.
2. Загрузка текста/фото/альбома.
3. Бот сохраняет один draft и показывает компактную панель действий.
4. Предпросмотр показывается только по кнопке 👁.

### Расписание
1. Выбор канала.
2. Показ текущего расписания или дефолтного шаблона.
3. Массовая загрузка медиа (одиночные фото и альбомы).
4. Завершение словом `готово`.
5. Распределение постов по ближайшим слотам.

## Настройки
- 📣 Каналы
- 🗓 Расписание каналов
- ✍️ Автоподпись

## Надёжность и миграции
- Безопасные миграции SQLite через `PRAGMA table_info` + `ALTER TABLE ADD COLUMN`.
- Таблица `post_batches` создаётся через `CREATE TABLE IF NOT EXISTS`.
- В `posts` поддерживается поле `batch_id`.

## systemd update flow
1. `git pull`
2. Обновить зависимости
3. Перезапустить systemd unit
4. Проверить `logs/bot.log`

## Proxy
- `PROXY_URL` и `AiohttpSession` не менялись.
